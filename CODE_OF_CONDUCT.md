# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, friendly, and respectful environment for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Be patient.
- Assume good intentions.
- Accept constructive criticism.

## Unacceptable Behavior

- Harassment
- Hate speech
- Personal attacks
- Trolling
- Spam
- Publishing private information

## Enforcement

Project maintainers reserve the right to remove comments, issues, pull requests, or contributors that violate this Code of Conduct.
